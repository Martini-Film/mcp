# Where This Repo Can Be Used

This repo is a public, stable home for Martini MCP integration metadata and docs.

## Direct Uses

| Use | How this repo helps |
| --- | --- |
| Gemini CLI extension gallery | Root `gemini-extension.json`, `GEMINI.md`, commands, policies, GitHub topic, and release tags. |
| Generic MCP client setup | README and `docs/clients.md` provide canonical install snippets. |
| Support handoff | `SUPPORT.md`, `SECURITY.md`, and public docs give users a clear path for help. |
| Release communication | `CHANGELOG.md` and GitHub releases track client-facing changes. |
| Partner review | Public docs explain auth, safety, and client support. |

## Provider Listing Collateral

| Platform | Role for this repo |
| --- | --- |
| Gemini CLI Extension Gallery | Primary Google/Gemini distribution path. |
| Claude Connectors Directory | Public setup/support/security documentation for the remote MCP submission. |
| Claude Plugin Directory | Can later grow a Claude plugin package that bundles Martini MCP with skills and slash commands. |
| ChatGPT Apps | Useful as public MCP/support docs, while any Apps SDK app can live separately. |
| Microsoft 365 Copilot connector review | Public documentation collateral if Martini pursues a read-only federated connector path. |
| Mistral or xAI custom MCP docs | Canonical client setup and safety documentation if they ask for public integration docs. |

## Not Included

This repo should not include:

- Product source code.
- MCP server implementation code.
- Private configuration.
- Internal provider prompts.
- Private test credentials.
