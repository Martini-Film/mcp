# Support

For Martini MCP support, email support@martini.film.

Include:

- MCP client name and version.
- Martini account email.
- Approximate time of the issue.
- Whether sign-in, connection, or a specific tool failed.
- Any visible error message.

Do not send API keys, access tokens, refresh tokens, or project assets by email unless Martini support specifically asks for a secure transfer method.
